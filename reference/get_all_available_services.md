# List supported service providers

List the names of all providers/vendors supported by the gptstudio chat
app.

## Usage

``` r
get_all_available_services()
```

## Value

A character vector

## Examples

``` r
get_all_available_services()
#> [1] "anthropic"    "azure_openai" "cohere"       "google"       "huggingface" 
#> [6] "ollama"       "openai"       "perplexity"  

```
