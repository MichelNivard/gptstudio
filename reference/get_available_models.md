# List supported models

Get a list of the models supported by the selected service.

## Usage

``` r
get_available_models(service)
```

## Arguments

- service:

  The API service

## Value

A character vector

## See also

[`get_all_available_services()`](get_all_available_services.md)

## Examples

``` r
if (FALSE) { # \dontrun{
get_available_models()
} # }
```
