# Chat UI

Chat UI

## Usage

``` r
mod_chat_ui(
  id,
  translator = create_translator(),
  code_theme_url = get_highlightjs_theme()
)
```

## Arguments

- id:

  id of the module

- translator:

  A Translator from
  [`shiny.i18n::Translator`](https://appsilon.github.io/shiny.i18n/reference/Translator.html)

- code_theme_url:

  URL to the highlight.js theme
